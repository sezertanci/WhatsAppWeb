# WhatsAppWeb

## Kullanılan Teknolojiler

- SignalR
- Onion Architecture CQRS Pattern
- .Net 6
- EntityFrameworkCore
- Mssql
- WebApi
- Automapper
- FluentValidation
- MediatR
- *DependencyInjections*
